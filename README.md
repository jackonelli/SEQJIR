# SEQJIR model for CoViD-19

SEQJIR-Model:

 S - Suspectible
 E - Exposed
 Q - Quarantine
 J - Isolated
 I - Infective
 R - Removed

## About

This project was made to fund a suitable model for estimating CoVid-19 outbreaks.
